1、分支结构 1、else if语句(多重if结构) 1、语法： if(条件1){ 语句块1; }else if(条件2){ 语句块2; }else if(条件3){ 语句块3; }.....else{ 语句块n //以上条件都不满足时，才运行的else(可有可无) } 2、switch-case结构 1、作用 为了简化 等值判断 的条件判断结构 2、语法 switch(变量/表达式){ case 值1: 语句块1; braek; //该语句块的结束 case 值2: 语句块2; break; ........... default: 语句块n; (break;) } 3、执行流程 1、解析 表达式或变量中的值 2、用 解析出的数值 与 case 后的数值进行等值比较 如果 比较成功，执行该 case 后的语句块 3、执行完成后，如果碰到break则结束整个switch 如果没有 break 从匹配的块开始向下 *执行（不会进行判断）*，至碰到第一个break或switch结束 4、switch-直落形式 两块或多块 case 之间，没有任何操作代码，表示 多块case要执行相同的操作 switch(变量/表达式){ case 值1: case 值2: case 值3: 语句块; break; } 2、循环结构：一遍有一遍的执行 相同或相似 的代码 1、特点： 1、循环条件 规定了循环的执行次数 2、循环操作 要执行相同或相似的语句-循环中做了什么 2、循环结构-while()循环 1、语法: while(条件){ //循环操作 } 2、流程 判断循环条件，条件为true,执行循环操作，至条件为false为止，跳出循环 3、 循环流程控制语句 1、continue; 终止本次循环的执行，继续下次循环 2、break; 提前退出整个循环结构 作用：可以帮助跳出死循环 3、循环结构-do...while()循环 1、语法 do{ //循环操作 }while(条件); 2、流程 先执行循环操作，再去判断循环条件，值为true，继续循环,值false,退出循环 3、作用： 最少可以运行一个循环体 4、 while() 循环与 do...while() 的区别 1、while()：先判断，后执行，最少一次都不执行循环操作 2、do...while()：先执行，后判断，最少执行一次循环操作 3、生成随机数(0~100) Math.random(); 会返回一个0~1之间的小数 猴子吃桃 第一天 吃了一半，又多吃了一个，第二天将剩下的桃子吃了一半 并且一个，以后每天都如此，第十天就剩一个
