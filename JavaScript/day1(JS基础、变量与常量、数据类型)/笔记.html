1、JS概述 1、什么是JS？ JS是一种运行与 JS解释器/引擎 中的解释型脚本语言 编译型语言：程序在运行前会进行编译 解释型语言：运行之前不会编译，直接执行，出错则停止 JS解释器/引擎：是JS脚本的运行环境 1、独立安装的JS解释器（NodeJS） 2、嵌入在浏览器内核中的JS解释器 PC，平板，手机，电视...全部都支持JS 2、 JS的发展史 1、1992年 Nombas 为自己的软件开发出一款脚本语言 C Minus Minus(C--) 后更名为 ScriptEase, 可以嵌入在网页中 2、1995年 Netscape为自己的 Navigator2.0 开发了另一种客户端脚本语言-LiveScript,为了借助 Java 的势头，改名为 JavaScript 3、1996年 Microsoft为了进军浏览器市场，在IE3.0中 发布了 JavaScript 的克隆版，成为JScript 4、1997年 Javascript1.1 作为草案提交给了ECMA(欧洲计算机制造商联合会)，从此JavaScript的核心，被更名成ECMAScript 完整的JS语言是由三部分组成： 1、核心（ECMAScript） 2、文档对象模型（DOM） Document Object Model,让JS有能力 对页面进行操作 3、浏览器对象模型（BOM） Browser Object Model, JS有能力 对浏览器进行操作 JS语言的特点和用途 1、特点： 1、纯文本，任何文本工具都可以开发JS 2、JS无需编译，直接由JS引擎来执行 3、弱类型语言 是由数据来决定数据类型 4、面向对象的语言 2、用途： 1、客户端的JS 1、客户端数据计算 2、表单输入验证 3、浏览器事件的触发和处理 4、网页特效制作 5、服务器端的异步数据提交（AJAX） 2、服务器端的JS 1、分布式运算 2、实时服务器 3、窗口应用 4、网络应用 2、JS基础语法 1、使用JS（重点） 1、浏览器内核（各类浏览器兼容性问题 的根本） 内核作用：负责页面内容的渲染 组成： 1、内容排版引擎-解析HTML/CSS 2、脚本解释引擎-解析JS 浏览器 内核名 排版引擎 脚本引擎 IE Trident - Chakra Firefox Gecko - Monkey Chrome webkit Webcroe V8 Safari webkit Webcroe Nitro Opera Presto - Carakan Opera(2013~至今) Webkit Webcroe V8 2、第一个JS程序 1、搭建JS运行环境 1、使用独立安装的JS解释器 console.log(""/''); 向控制台打一段脚本代码并执行 2、使用浏览器内核中的JS解释器 1、直接在浏览器的Console中输入脚本并执行 2、将JS脚本嵌入在HTML页面中执行 1、HTML元素事件编写脚本 事件：onclick 语法： 2、在元素（可以出现在HTML中任意位置）中嵌入JS脚本 3、使用外部的脚本文件编写JS脚本 1、创建一个 **.js 文件 在文件中编写JS脚本 2、引入JS文件 2、JS调试（重要） 1、代码错误 解释型语言，若某行代码出现错误，解释器则终止此次执行，但不会影响后续快和后续的HTML执行 3、语法规范 1、编写JS代码 1、语句-会被JS解释器所执行的代码 console.log(); document.write(); 由表达式、关键字、运算符组成 大小写敏感 userName 与 Username 不同 大多数由分号或换行表示结束 2、注释-不会被JS解释器所执行的内容 单行注释:// 多行注释：/* */ 3、变量与常量 1、什么是变量 问题：向银行存1000元，每年利率5%，10年后本息 内存：保存计算机程序在运行过程中所需要用到的数据（临时存储） 变量：内容中的一段存储空间，目的是为了临时的保存数据 值：保存在变量（内存空间）中的数据 变量名：内存空间的别名，可以自定义 2、变量的声明：到内容中申请一段空间保存数据 1、语法： 1、声明变量 var 变量名1; 2、为变量赋值 变量名=值; 注意： 声明时可以省略 var 关键字，省略之后变量就成了“全局变量”，推荐不要省略 var 关键字 ex:声明一个变量叫 age，保存数据 18 var age; age=18; 3、声明变量并直接赋值 var 变量名=值; ex: var age=18; 4、在一条语句中声明多个变量 声明 age1值为18，age2值为36,age3值为50 ex:var age1=18,age2=36,age3=50; 2、变量名命名规范 1、不允许使用JS关键字及保留关键字 2、可以包含字母、数字、下滑线_以及$ 3、变量名不能以数字开头 4、变量名最好见名知意 5、可以采用"匈牙利命名法"，"驼峰命名法"，"下划线命名法" 驼峰命名法：如果变量名由一个单词组成，全小写， 如果变量名由多个单词组合而成，第一个单词小写，之后单词的首字母大写 3、变量的使用 1、未经初始化的变量 声明了变量，但是没有被赋过值，默认值为 underfined（未定义） 小习惯： 声明变量时，如果字符串的话可以设置默认为""，如果是数字的话，可以设置默认值为0 2、变量未被定义过，而被直接使用，语法错误 3、对变量值进行的存、取操作 1、对变量进行存值操作-SET操作 运算符: =（赋值运算符） 如果变量名出现在 = 的左边，就是赋值操作 2、对变量进行取值操作-GET操作 只要变量名称没有出现在=的左边，那么就都是取值操作（只要不是赋值的，那就都是取值） 4、运算符-算术运算符 +,-,*,/,%(模) %:取余数 使用场合： 1、判断奇偶性和倍数 2、获取某数字的最后几位 +： 1、数字运算 +两边的数据 都是数字类型，则进行的就是 加法操作 2、字符连接 如果 +两边有一个是字符串(""/'')的话，那么进行的就是 字符连接 操作 5、常量的声明和使用 1、什么是常量 在程序中，一旦声明好，在运行过程就不允许被修改的数据，称之为常量 常量在程序中表示的是常理型的数据，如：PI（π） 2、语法 const 常量名称=值; ***注意：常量名一般采用全大写的模式*** 4、数据类型 8bit（位）=1byte（字节） 1024byte=1KB（千字节） 1024KB=1MB（兆字节） 1024MB=1GB 1024GB=1TB 问题：保存一个数字（25）到内存中，占据多大空间 1、什么是数据类型 保存在内存中的数据的类型，数据类型决定了数据在内存中占据多大的空间 2、数据类型分类 1、基本数据类型（原始数据类型） 1、number类型 可以表示32位(4byte)的整数，也可以表示64位(8byte)的浮点数(小数) 整数： 10进制：逢10进1 8进制：逢8进1 以0作为开始的数组都是8进制数 16进制：逢16进1 0~9，A~F 以0x作为开始的数字都是16进制 小数： 小数表示法：使用小数点记录数据：12.34 指数表示法：45186e33 2、string类型（字符串类型） 表示一系列的文本字符数据,每个字符 都有一个 独立编码 由Unicode字符、数字、标点符号组成 汉字在Unicode编码的范围：("\u4e00")~("\u9fa5") 查看字符的Unicode编码（默认10进制） 字符.charCodeAt(); 查看(2)进制 字符.charCodeAt().toString(2); 要求：字符串使用过程中国，必须用""/''引起来 转义字符： 1、\n:换行 2、\r:回车（非换行，接受指令） 3、\t:缩进 4、\':' 5、\":" 6、\\:\ 3、boolean类型（布尔类型） 来表示结果的真或假/肯定或否定 该类型仅有2个值： true 表示真（肯定） false 表示假（否定） ex: var 变量名=true; 在实际运算中 true 可以当成1做运算, flase 可以当成0做运算 4、underfined类型（） 访问的数据不存在 产生underfined的场合： 1、声明变量未赋值 2、访问 对象的属性不存在 2、引用类型
