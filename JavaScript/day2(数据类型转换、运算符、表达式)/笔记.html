1、数据类型的转换 1、弱类型：由数据决定 数据类型 var num; //underfined num=13; //number num="你好" //string num=false; //boolean 强数据类型：由数据类型决定数据 1、typeof()函数 作用：显示指定数据的数据类型 语法： typeof(变量)/typeof 变量; 2、隐士转换:不同类型的数据早计算过程中会自动进行转换 特点： 1、任何类型的数据，在与string做"+"运算时，都会转换成 string NaN:Not a Number consloe.log(isNaN()); 判断是否为 非数字类型（只判断数值与类型无关）,是数字返回false,不是数字返回true 3、强制转换（显示转换）:由JS提供的一组转换函数来完成类型的转换 1、toString(); 转换成字符串，任一数据类型的数据均可转换成string类型 语法：数据(变量).toString() 2、parseInt(数据); 将数据转换为number类型（整数） 碰到第一个非整数字符，则停止转换 如果无法转换时，转换的结果为NaN 3、parseFloat(数据); 将数据转换为number类型（小数） 碰到第一个非小数点或数字字符，则停止转换 4、Number(数据); 将一个数据解析为Number类型 ***注意：如果数据中包含非法字符(数字/小数点),则返回NaN*** 2、运算符和表达式 1、运算符：做数据的运算 2、表达式：由运算符和操作数所组成的式子，称之为表达式 var a=5; //赋值表达式 var b=10; //赋值表达式 a+b; //算术表达式 注意：每一个表达式，都有值，而且计算顺序是从右到左 3、运算符的分类 1、算术运算符 +,-,*,/,% 注意： -,*,/,% 如果两边的操作数不是number类型，会试图通过 Number() 先进行转换，如果转换不成功，结果为NaN 优先级：先*,/,%再+,- 提升优先级：加 () 2、关系运算符 1、运算符 <,>,<=,>=,==(判断等于),!= ===(全等),!===(全不等) === 与 == 区别： 1、== 只判断数值，不判断类型 2、=== 既要判断数值也要判断类型 由关系运算符组成的表达式称之为关系表达式，关系表达式的结果都是boolean类型 2、特殊注意： 1、字符串 和 数字 比较大小关系时，先将字符串通过Number()转换成number类型，再进行比较 2、字符串 和 字符串 比较大小时，是比较每位字符的Unicode编码 3、NaN参与boolean比较时，结果一律是false 3、逻辑运算符：多条件之间的判断关系 1、逻辑非:! 对现有条件结果进行取反 2、逻辑与:&& 判断两个条件，如果两个条件的值，都是true，表达式结果为true,有一个false，表达式结果为false 3、逻辑或:|| 判断两个条件，两个条件的值，只要有一个为true，表达式结果为true 4、短路逻辑 1、&&(短路与) 如果第一个表达式的结果为 false,那么第二个表达式就不用判断，整个表达式的结果为 false 如果第一个表达式的结果为 true,则继续判断第二个表达式的结果，并以第二个表达式的结果作为整个表达式的结果 2、||(短路或) 如果第一个表达式的结果为 true,就不在判断第二个表达式，并且将 true，作为整个逻辑的结果 如果第一个表达式的结果为 false,则继续判断第二个表达式，并且将第二个表达式的结果，作为整个逻辑的结果 4、位运算（运算快） 1、特点： 1、做数字运算 2、将数字转换成2进制，再进行运算 2、运算符 1、&(按位 与) 将两边的操作数转换成2进制，比较每位的数字，只有对应都为1的时候，该位的结果才为1，否则结果就是0 作用：判断一个数字的奇偶性，判断的数字与1做按位与，结果是1的话，数字是奇数（数字&1） 2、|(按位 或) 将两边的操作数转换成2进制，比较每位的数字，两个数中有一个是1的话，该位的结果就是1，否则结果就是0 作用：向下求整（数字|0） 3、<<(按位 左移) 将2进制数向左移动几位，后面以0补齐 4、>>(按位 右移) 将2进制数向左移动几位，前面以0补齐 5、^(异或) 将两边的数字，转换成2进制进行比较，每位上的数字，只有一个为1时，该位结果才为1，否则0 作用：可以使两个数字做交换 ************************************ 弹出提示框:window.alert(); 弹出输入框:window.prompt(); 舍入误差问题：.toFixed(x); 保留X位小数 isNaN() 函数 判断数据是否是NaN ************************************
