1、运算符 1、赋值运算符 1、"=" 计算顺序从右到左，赋值符号的左边，必须是变量 2、扩展赋值表达式 +=,-=,*=,/=,%=,&=,|=,^= ex: var num=5; num+=3 ==> num =num + 3; 语法：a+=b; ==>a=a+b; 2、字符串连接:"+" 3、条件运算符(三目(元)运算符) 单目运算符：!,++,-- 双目运算符：+,-,*,/,%,&&,||,^,&,|,<,> 三目运算符：需要三个操作数/表达式 语法： 表达式1?表达式2:表达式3; 表达式1的运行结果应为boolean类型 若表达式1为 true,则整个表达式的值为表达式2的运算结果 反之，则整个表达式的值为表达式3的运算结果 条件表达式允许被嵌套： 判断成绩，>=80优秀,>=60合格,否则输出不及格 score>=80?"成绩优秀":(score>=60?"合格":"不及格"); 2、函数 1、什么是函数(function):可以被称之为方法(method),过程(procedure) 它是一段预定好，并可以反复使用的代码块 1、预定义：事先声明好的内容，不会马上被执行 2、反复使用：提升代码的复用性 3、代码块：允许包含若干条的执行语句 其实函数就是一段独立的功能体，同时也是一个功能完整的对象 没有函数的弊端： 1、事件中想执行JS脚本只能逐行编写 2、可复用性不高 2、定义函数 语法：function 函数名(参数列表){ 待执行的代码块 } 调用函数：函数名(); 在任意合法的JS脚本位置处都可以调用函数 3、带参数函数的声明 1、参数： 定义函数，需要由外部传入到函数内部进行辅助运行的数据，称之为参数 console.log("Hello World"); 2、声明带参数函数：function 函数名(参数1,参数2,参数3){} 参数列表：由","来隔开的参数名称的声明 定义函数时所定义的参数称之为"形式参数" 调用带参数函数 函数名(参数列表); 参数列表：由","隔开的具体参数值来组成的列表 调用函数时所传递的参数称之为"实际参数" 4、带返回值的函数声明 1、返回值：调用完函数后，由函数内部返回给函数调用处的一个值 2、声明带返回值的函数 function 函数名(参数列表){ //待执行的语句 return 值; } 注意： 1、方法的返回值可以灵活添加，但最多只能有一个 2、程序碰到return就会返回到函数调用处，所以，return后面的代码就不会被执行的 5、函数内调用函数 function 函数(参数列表){ 函数(); } 6、变量的作用域 1、作用域：就是变量与函数的可访问范围，它控制着函数与变量的可见性与生命周期 JS中作用域分为2种 1、函数作用域，只在当前函数内可访问 2、全局作用域，一经定义，在代码中的任何位置都能访问 2、函数作用域中的变量 也叫 局部变量，只能在他所在 函数 中使用 3、全局作用域中的变量 也叫 全局变量，定义后在，JS的任何位置处都可以访问的到 定义方式: 1、独立于任何function的位置处（就是与function同级定义） 2、声明变量时不使用var关键字 注意：必须先调用 变量所在的函数后，才可以继续使用全局变量 4、声明提前 JS程序在正式运行前，会将所有的var声明的变量和function声明的而函数，预读到所有作用域的顶部， 但是不会对其赋值，对变量的赋值还保留在原来位置 5、按值传递 基本数据类型的数据，在作为实际参数传递时，实际上将参数的值赋值了一份副本进行传递的 7、函数作用域 也分为“全局函数”以及“局部函数” 全局函数： 声明在独立于任何function的位置处，位于JS的最外层 局部函数： 在一个函数内部定义的函数，就是局部函数 3、分支结构 1、程序的流程控制 程序=数据结构 + 算法 任何复杂的算法都有以下几种结构： 1、顺序结构 2、分支选择结构 程序在运行过程中，在不同的条件下可以选择执行某些程序 1、当条件满足时运行某些语句 当条件不满足时则不运行这些语句-if结构 2、当条件不满足时运行某些语句 当条件不满足时-else结构 3、循环结构 2、if结构 1、语法：if(条件){ 语句块 } 执行流程： 1、判断条件结果 若值为true,执行if语句块中语句 2、注意 1、条件应该是boolean类型的值或表达式 如果条件不是boolean的话，则会进行自行转换 以下几情况会默认转换为false if(0){} if(0.0){} if(NaN){} if(undefined){} if(""){} if(null){} 2、if(条件)后的{}可以省略，如果省略了{}的话，那么if结构将只控制下面的一个语句 3、if..else 结构 1、语法： if(条件){ 语句块1 } else{ 语句块2 } 执行流程： 值为true,执行语句块1 值为false,执行语句块2
